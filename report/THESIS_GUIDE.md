# Агент суурилсан хиймэл оюуны микросервис архитектур - Дипломын ажил

## Тойм

Энэхүү дипломын ажил нь 4 дүгээр курсын Мэдээллийн технологийн оюутны хувьд бичигдсэн ба "AI Agents for Microservices" (Агент суурилсан хиймэл оюун ухааны микросервис архитектур) сэдвээр боловсруулсан болно.

## Бүтэц

Дипломын ажил нь дараах бүлгүүдээс бүрдэнэ:

### 1. Удиртгал
- Хиймэл оюуны инженерчлэлийн өсөлт
- Микросервис архитектурын давуу тал болон сорилтууд
- Судалгааны зорилго

### 2. Онолын хэсэг
- **Хиймэл оюун инженерчлэлийн үндэс**
  - Хиймэл оюун инженерчлэл гэж юу вэ
  - Суурь загваруудын хөгжил (Language Models → LLMs → Foundation Models)
  - Суурь загваруудын сургалт (Pre-training, Post-training)
  - Загваруудын үр дүнг хэмжих

- **Prompt инженерчлэл**
  - Prompt бичих шилдэг арга барил
  - Sampling стратегиуд (Temperature, Top-k, Top-p)
  - Хамгаалалтын prompt инженерчлэл

- **Retrieval-Augmented Generation (RAG)**
  - RAG хэрэгтэй болох шалтгаан
  - RAG системийн бүтэц
  - Retrieval алгоритмууд (Term-based, Embedding-based)
  - RAG-ын үнэлгээ ба сайжруулах аргууд

- **Хиймэл оюун агентууд**
  - Агент гэж юу вэ
  - Агентын бүрэлдэхүүн хэсгүүд (Орчин, Үйлдлүүд, Даалгавар)
  - Багажууд (Tools)
  - Төлөвлөлт (Planning)
  - Эргэцүүлэн бодох ба алдаа засах
  - Агентын санах ой

### 3. Микросервис архитектур
- Микросервис гэж юу вэ
- Микросервисийн давуу тал
- Микросервисийн сорилтууд
- Микросервис хоорондын харилцаа

### 4. Асуудлын тодорхойлолт
- Сервис хоорондын нарийн төвөгтэй логик
- Өгөгдлийн нэгтгэл ба шинжилгээ
- Динамик routing ба шийдвэр гаргалт
- Мониторинг ба алдаа илрүүлэлт
- Хиймэл оюун агентууд хэрхэн туслах вэ

### 5. Шийдэл ба санал болгож буй загвар
- Хиймэл оюун агент суурилсан микросервис архитектур
  - Orchestrator Agent
  - Service Agents
  - Knowledge Agent
  - Monitoring Agent
- Системийн архитектур ба workflow
- Техникийн нарийн ширийн зүйлс
- Санал болгож буй загварын давуу тал

### 6. Туршилт ба кодын жишээ
- Прототип систем (E-commerce жишээ)
- Orchestrator Agent хэрэгжүүлэлт (Python код)
- Knowledge Base хэрэгжүүлэлт (RAG систем)
- Service Agent хэрэгжүүлэлт
- Туршилтын үр дүн
- Харьцуулалт (Агент vs Уламжлалт систем)
- Хэлэлцүүлэг

### 7. Дүгнэлт
- Онолын болон практик үр дүн
- Цаашдын хөгжил
- Судалгааны ач холбогдол
- Эцсийн дүгнэлт

### 8. Ном зүй
- Chip Huyen-ийн "AI Engineering" ном
- Суурь загварын судалгаа (GPT, BERT, Llama)
- RAG болон AI Agents-ын судалгаанууд
- Микросервисийн архитектурын ном зохиолууд
- Vector databases ба embedding-ийн судалгаанууд

## Кодын жишээнүүд

Дипломд дараах Python кодын жишээнүүд багтсан:

1. **Orchestrator Agent Implementation** - Агентын үндсэн логик
   - Request processing
   - Context building
   - Plan generation with LLM
   - Plan execution
   - Error handling
   - Response generation

2. **Knowledge Base Implementation** - RAG систем
   - Document indexing
   - Embedding generation with Sentence-Transformers
   - FAISS vector search
   - Top-k retrieval

3. **Service Agent Implementation** - Микросервис дуудлага
   - Service discovery
   - HTTP API calls
   - Error handling
